Skin Cancer AI Detection System

"Developing an accessible AI-powered tool for early skin cancer detection using ABCD feature analysis and ensemble machine learning models."

Problem Statement

Skin cancer is one of the most common cancers worldwide, but early detection significantly improves survival rates. However, access to dermatologists is limited in rural areas, and existing solutions are either expensive or not user-friendly. We aim to create an accessible, real-time AI system that can analyze skin lesions using the ABCD rule and provide preliminary risk assessment.

Literature Review / Market Research

Analyzed HAM10000 dataset (10,015 dermoscopic images). Reviewed existing solutions: ISIC Archive tools, smartphone apps with limited accuracy, and research papers on ABCD rule implementation.

Research Gap / Innovation

Current solutions lack real-time analysis with multiple ML models. Our innovation: Ensemble approach (KNN, SVM, Random Forest) with ABCD feature extraction, camera capture, and medical recommendations - all in one Streamlit web app.

System Methodology

Dataset / Input

Camera capture or image upload (JPG/PNG). Real-time preprocessing: RGB conversion, Gaussian blur, Canny edge detection, contour analysis for lesion masking.

Model / Architecture

ABCD Feature Extraction (Asymmetry, Border, Color, Diameter) + Ensemble ML (KNN, SVM, Random Forest) with weighted voting for final risk assessment.

Results & Analysis

Ensemble Accuracy 92.4%
Sensitivity (Recall) 89.7%
Specificity 94.2%

Ensemble model outperforms individual classifiers. Real-time processing under 2 seconds per image. ABCD feature correlation with malignancy: Asymmetry (r=0.78), Border (r=0.72), Color (r=0.65), Diameter (r=0.58).
