Skin Cancer Detection using ML

"Developing an accessible AI-powered tool for early skin cancer detection using ABCD feature analysis and ensemble machine learning models trained on HAM10000 and ISIC datasets."

Problem Statement

Skin cancer is one of the most common cancers worldwide, but early detection significantly improves survival rates. Access to dermatologists is limited in rural areas, and existing solutions are either expensive or not user-friendly. This project creates an accessible, real-time AI system that analyzes skin lesions using the ABCD rule and provides preliminary risk assessment through a web-based interface.

Datasets Used

HAM10000 dataset — 10,015 dermoscopic images across 7 diagnostic categories. ISIC 9-Class dataset from Kaggle — additional diversity across 9 skin lesion categories. Combined training set: 12,372 images after merging.

Research Gap / Innovation

Current solutions lack real-time analysis with multiple ML models. Innovation: Ensemble approach (KNN, SVM, Random Forest) with ABCD feature extraction, probability-based thresholding, image quality validation, and medical recommendations — all in one Streamlit web app.

System Methodology

Data Pipeline

HAM10000 + ISIC images resized to 28×28, normalized to [0,1], flattened to 2,352 features. Class balancing via undersampling: 6,000 balanced training samples (3,000 benign + 3,000 malignant).

Model Architecture

Three ML classifiers: KNN (k=5), SVM (RBF kernel, C=1.0), Random Forest (200 trees, max_depth=15). Soft voting ensemble averages malignant probability across all 3 models for final prediction.
